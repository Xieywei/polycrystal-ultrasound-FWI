# polycrystal-ultrasound-FWI
Data, models, and scripts for the paper "Subsurface polycrystalline microstructure reconstruction via immersion ultrasound full waveform inversion"

# Subsurface Polycrystalline Microstructure Reconstruction via Immersion Ultrasound Full Waveform Inversion

This repository contains the datasets, scripts, and supplementary materials associated with the paper:

**“Subsurface polycrystalline microstructure reconstruction via immersion ultrasound full waveform inversion”**  
*Author: Yongwei Xie, Shuguang Fan, Jiaze He, Yanju Liu, Jinsong Leng
*Affiliation: Department of Astronautical Science and Mechanics, Harbin Institute of Technology, Harbin, 150001, China.
*Year: 2025*

The provided materials allow users to explore the immersion ultrasound FWI workflow, reproduce selected results from the paper, and use the dataset for further research.
